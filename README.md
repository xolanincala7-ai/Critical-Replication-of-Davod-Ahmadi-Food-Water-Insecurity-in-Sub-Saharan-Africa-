# Critical-Replication-of-Davod-Ahmadi-Food-Water-Insecurity-in-Sub-Saharan-Africa-
Critical replication of Davod Ahmadi’s 2019 study on food and water insecurity in Sub‑Saharan Africa, using Afrobarometer Round 6 data and RStudio to reproduce and improve the original analyses — my strongest replication project in the MA‑E‑Science programme.
